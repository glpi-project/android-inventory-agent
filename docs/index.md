## Install the app

To use a configuration deep link, you need the **GLPI Android Inventory Agent** installed on your device.

[Download on Google Play](https://play.google.com/store/apps/details?id=org.glpi.inventory.agent){: .btn}

---

## What is GLPI Android Inventory Agent?

The GLPI Android Inventory Agent collects hardware and software inventory from Android devices and sends it to your [GLPI](https://glpi-project.org/) server — helping IT teams keep track of all mobile assets automatically.

### Features

- Automatic hardware inventory (CPU, memory, storage, battery…)
- Installed application listing
- Network information (Wi-Fi, Bluetooth, SIM)
- Remote configuration via deep link
- Scheduled or manual inventory submission
- Compatible with GLPI 10+
